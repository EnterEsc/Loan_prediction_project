# Кредитный скоринг — предсказание одобрения кредита

## Описание проекта
Веб-приложение для предсказания одобрения кредита на основе данных о заёмщике.  
Модель обучена на датасете из 4269 заявок.

## Как запустить

1. **Клонируйте репозиторий**
2. **Установите зависимости**:
   ```bash
   pip install -r requirements.txt
3. **Запустите приложение**:
    ```bash
    streamlit run app.py
